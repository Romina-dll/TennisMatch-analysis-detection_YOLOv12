# Tennis Match Analysis & Object Detection with YOLOv12

> An end-to-end computer vision project for automated **tennis match analysis** using the state of art **YOLOv12 object detection framework**. The system is designed to detect key entities in tennis match videos, including **players**, **the ball**, and **court-related elements** and enabling visual analysis and future performance analytics.
